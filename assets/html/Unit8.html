2D Arrays:
This unit comes with its own FRQ question: question 4.
According to college board, question 4 will ask about, Students will be asked to write program code to satisfy method specifications using expressions, conditional statements, and iterative statements and create, traverse, and manipulate elements in 2D array objects.
2D arrays are multidensional arrays and are created by:

Type[][] 2dArray = new Type[][];

When updating values in the array you have to use a specific way. It goes by row, followed by column.

array[row][col] = value;
