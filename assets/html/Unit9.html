Inheritance:
This is personally my favorite unit, and the unit I seem to be the most consistent in.
Inheritance is the relationship between classes, and allows for many similar classes to be created more efficently.
You can create the subclass relationhsip to the parent class by:

class SubClass extends ParentClass {}

The subclass will then inherit all of the parent's classes attributes/methods.
The subclass is also able to relate to the parent class by using the super keyword, which allows the subclass to call the parent classes methods and constructor.
Polymorphism:
I made a lesson on this :)
Polymorphism allows subclasses to override parent classes methods and ultize them at run time. 

class ParentClass{
    public void greet(){
      System.out.println("hello");
    }
}

class SubClass extends ParentClass{
    public void greet(){
      System.out.println("hi");
    }
}

ParentClass myClass = new SubClass();
myClass.greet(); //outputs hi
