The main part of this unit is using object methods, especially those of common classes like String.
A very tricky method is the String.substring() method. It has 2 variations:

  • substring(int index)
  • substring(int start, int end)

    • The difficlt part of these two methods is the when an IndexOutOfBounds Exeption is thrown.
    For example:

  • ("Hello").substring(5, 6) - causes an error
  • ("Hello").substring(5) - does not

    • This is because the first expression asks the string for the character at the 5th index. While the second expression does not.

    During the Ap exam we will have a reference sheet with all of the methods; but it is still good to remember how each method works.
