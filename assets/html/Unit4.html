Iteration:
There are many different ways to iterate over values:

  • while(condition) - while loops, repeats whats on the inside while the condition remains true. 
      int t;
while((t=streamReader.read())!=-1){
    //do stuff
}
      this is a fairly common format for reading InputStreams
  • for(condition) - for loops, generally used for iterating a certain number of times 
      for(int i=0; i<10; i++){
    //do stuff
}
      iterates from 0-9 
      for(Type v : array){
    //do stuff
}
      iterates through each value in an array

    • Nested Iteration:
    For each step in an outer loop, the inner loop does all of its steps 

  • while(condition){
    //for each outer loop
    for(int i=0;i<10;i++){
        //inner loop iterates through 1-10
    }
}

    • During the Ap exam a lot of questions rely on an understand of iteration. Some even have you follow the execution of the loops yourself to determine what they do.
