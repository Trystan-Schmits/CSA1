Boolean expressions:
These are generally found contained within if statements, and are used to check conditions. There are some basic boolean statements like:

  • == - equals
  • != - not equal
  • > - greater than
  • < - less than
  • >= - greater than or equal to
  • <= - less than or equal to

    • And some more logical operations like:

  • ! - not
  • && - and
  • || - or

    • De Morgan's Law:
    Basically if a "not" statement is used outside of a block, it is equivical to everything inside the block flipping.

    !(a && b) == !a || !b
    !(a || b) == !a && !b

    Comparing Objects:
    References are compared using:

    reference == reference

    null comparisons should be used with a reference check.
    If you want to compare if two objects by their values you should use comparison methods.

  • Object.equals() - most common, every object (arrays, classes) includes this method since it is inherited from the object class
  • Object.compareTo() - returns an int with the result of the comparison, not included on every class by default, but still fairly common
