Writing Classes:
This unit comes with one of the main frq questions, Question 2. The purpose is to build classes, a common way to structure and create Objects.
The FRQ has you write out a class, or a part of a class, with proper structure and functionality. It is important to pay attention to the different sections of the class as you are writing it. These sections include:

  • Class Header
  • Constructor
  • Instance Variables
  • Methods
