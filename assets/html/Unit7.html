Array Lists:
Array Lists are very similar to arrays, and share an frq question: question 3.
Array lists can vary in size during runtime, this means that you can both add and remove values in the array list while the program is running. Furthermore is comes with some extra methods such as arrayList.contains(Object value)

The FRQ question generally has you create methods that utilize either arrays or arraylists to serve the scenarios purpose
For example a list of comments could be stored in an array, and you have to find which comments contain some sort of pattern and remove them.
Array Lists:
There are a few differences between arrays and arrayLists:

  • array.length - size of an array
  • arrayList.size() - size of an arrayList
  • array[index] - getting a value at a given index in an array
  • arrayList.get(index) - getting a value at a given index in an arraylist
