Arrays:
Arrays are the most basic form of a list that java has (without using the list class).
They are objects that store values at different indexes. These values can be accessed by using the indexes.
When creating an array you specify the type and the size of the array. The array cannot change size dynamically. If you want to change the size of the array, you must create a new one.
Arrays can be initailized by:

  • Type[] array = new Type[size]; - creates an empty array with the given type and size
  • Type[] array = {value1, value2, ...}; - creates an array with the given information

    • Values in the arrays can be accessed by:

    array[index]

    Iterating through Arrays:
    this can be done in two major ways:

  • for(int i=0; i< array.length; i++) - use a for loop to get each index in the array
  • for(Type value : array) - use a for loop to get each value in the array

    • You could also use other kinds of iteration like while loops and recursion to do this as well.
